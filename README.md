# Super-Trunfo-em-C-Estacio
Olá! Neste projeto estarei desenvolvendo um jogo Super Trunfo na linguagem C para a faculdade.
O projeto está dividido em três etapas: "Nível Novato", "Nível Aventureiro" e "Nível Mestre". 
Cada nível possui uma adição de informações e comandos para que possamos desenvolver a criação de códigos na prática.
